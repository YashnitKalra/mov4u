# What it does:
You can search a movie, know its ratings, plot, watch trailer...and other information or you can also view top 100 rated and popular movies.

# Made Using:
1. HTML
2. CSS
3. Bootstrap 4
4. jQuery
5. PHP
6. TMDB API is used to fetch movies information.
